# Healthcare Data Governance & Security

## Overview
This section demonstrates engineering best practices for handling sensitive
healthcare data in a HIPAA-aligned environment.

> Note: This focuses on technical safeguards, not legal certification.

## Key Concepts
- PHI vs Non-PHI data classification
- Least-privilege access control
- Encryption at rest and in transit
- Column-level masking
- Audit logging and lineage

## Security Practices
- IAM-based access policies
- Secrets management
- Secure data isolation

## Why This Matters
Strong governance ensures patient privacy, regulatory compliance,
and stakeholder trust.
