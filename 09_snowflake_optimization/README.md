# Snowflake Performance & Cost Optimization

## Overview
This project demonstrates best practices for building efficient and
cost-aware data models in Snowflake.

## Techniques Used
- Star schema design for analytics
- Clustering for partition pruning
- Query refactoring to reduce scans
- Warehouse sizing and auto-suspend

## Cost Optimization
- Separate compute for ingestion and analytics
- Credit usage monitoring
- Query profiling

## Outcome
- Faster query performance
- Improved concurrency
- Reduced warehouse costs
